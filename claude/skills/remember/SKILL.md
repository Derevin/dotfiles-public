Save the current task context to your auto-memory files so we can pick up where we left off in a future session.

Summarize and save:
1. What we were working on (the problem/goal)
2. Current status (what's been done, what's pending)
3. Key findings and decisions made
4. What to test or do next

Write this to the appropriate memory file. Keep it concise — enough to resume, not a full transcript.
